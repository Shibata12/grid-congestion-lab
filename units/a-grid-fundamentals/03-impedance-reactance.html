A-03: インピーダンスとリアクタンス

A-02: 潮流計算の基礎 では、ループ系統の潮流が リアクタンス（電気的な距離）に反比例して分配されると学びました。 では、そのリアクタンスとは何でしょうか。本ユニットでは交流回路の視点から、 抵抗・リアクタンス・インピーダンスの関係を概念的に整理し、 なぜ送電線ではリアクタンスが主役になるのかを説明します。A-02で“預けて”あった概念を、ここで回収します。

このユニットの範囲について： 本ユニットは概念的な理解を重視し、複素数を使った厳密な計算には踏み込みません。 数式は考え方を示すために最小限だけ用います。また、送電線の潮流の流れにくさとして主に効く 誘導性リアクタンス（コイル）を中心に扱います。

1. 直流と交流：「流れにくさ」の正体

A-01・A-02では、オームの法則やキルヒホッフの法則を直流的に扱ってきました。 直流（DC）では、電流の向きと大きさが一定です。このとき電流の流れにくさを決めるのは、 抵抗電流の流れにくさのうち、電気エネルギーを熱に変えて消費する成分。記号 R、単位はオーム（Ω）。 R だけです。

ところが電力系統は交流（AC）です。交流では電流の向きと大きさが周期的に変化します （日本では東日本50Hz・西日本60Hz）。電流が「変化する」と、抵抗とは別の流れにくさが現れます。 それがリアクタンスです。A-02でループ系統の潮流分配を「リアクタンスで決まる」としたのは、 系統が交流だからにほかなりません。

2. リアクタンス：変化を妨げる流れにくさ

リアクタンスの正体は、コイル電線を巻いたもの。インダクタンス L を持ち、電流の変化を妨げる。送電線も電線が長く伸びており、インダクタンスを持つ。（インダクタンス L）が持つ 「電流の変化を妨げる」性質です。コイルに流れる電流を増やそうとすると、それを打ち消す向きの起電力が生じて妨げ、 減らそうとすると、電流を維持しようとして妨げます（自己誘導）。

ここがポイントです。直流の定常状態では電流が変化しないので、コイルはただの導線で、リアクタンスは効きません。 ところが交流では電流が常に変化し続けるので、コイルは流れにくさとして働き続けます。これが リアクタンス交流で電流の変化を妨げる成分の流れにくさ。記号 X、単位はオーム（Ω）。コイル（インダクタンス）に生じる誘導性リアクタンスが代表。 X です。 そして、電流の変化が速いほど（＝周波数 f が高いほど）妨げは強くなり、リアクタンスは大きくなります。

（この式は「周波数が高いほどリアクタンスが大きい」という関係を示すためのもので、本ユニットで計算は行いません。）

抵抗との大切な違いも押さえておきましょう。抵抗は電気エネルギーを熱に変えて消費します。 一方、理想的なリアクタンス（コイル）はエネルギーを磁界としていったん蓄え、また回路へ返すので、 それ自体は電力を消費しません。この「消費されない電力」は無効電力と呼ばれ、A-04: 有効電力・無効電力と電圧で扱います。 なお、コンデンサにも容量性のリアクタンスがありますが、送電線の潮流の流れにくさとして主に効くのは、ここで見た誘導性リアクタンスです。

3. インピーダンス：抵抗とリアクタンスの合成

交流回路全体の流れにくさを表すのが インピーダンス交流回路における電流の流れにくさの総合指標。抵抗RとリアクタンスXを直角な2成分として合成したもの。記号 Z、単位はオーム（Ω）。 Z です。 これは抵抗 R とリアクタンス X を合わせたものですが、単純な足し算ではありません。 R と X は「効き方の向き」が違うため、直角に交わる2つの成分として合成します。 図にすると直角三角形になり、その斜辺がインピーダンスです（インピーダンス三角形）。

たとえば抵抗 R = 3 Ω、リアクタンス X = 4 Ω のとき、インピーダンスは Z = √(3² + 4²) = √25 = 5 Ω です。単純に足した 3 + 4 = 7 にはならない点が重要です。

なぜ「直角」なのかというと、交流では抵抗にかかる電圧とリアクタンスにかかる電圧とで、 変化のタイミング（位相）がちょうど90°ずれるためです。 ここでは「向きが違うので直角に合成する」とだけ捉えておけば十分です。

4. なぜ送電線ではリアクタンスが支配的か

ここで本題、A-02の裏付けです。高電圧の送電線では、リアクタンス X が抵抗 R よりずっと大きいという特徴があります （X ≫ R、いわゆる X/R 比が大きい）。

すると、さきほどのインピーダンス三角形は縦に細長くなり、斜辺 Z はほぼ縦の辺 X と同じ長さになります。 つまり Z ≈ X。インピーダンス（流れにくさ）は、ほとんどリアクタンスで決まってしまうのです。

だからこそ A-02: 潮流計算の基礎 では、経路の流れにくさを 「主にリアクタンスで決まる」とし、リアクタンスを「電気的な距離」とみなして、 潮流はリアクタンスに反比例して分配されると扱うことができました。抵抗を主役から外せたのは、送電線で X ≫ R だからです。

この性質は、のちの D-01: PTDFと感度分析 でも使われます。 そこでは抵抗を無視し、リアクタンスだけで潮流分配を近似します（PTDFがリアクタンス分配率に等しくなるのはこのためです）。 まとめると、抵抗は熱損失や配電系統では効きますが、基幹系統の潮流分配の主役はリアクタンスだ、ということです。
