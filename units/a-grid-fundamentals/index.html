A: 系統の基礎

この領域では、電力系統を理解するための物理的な土台を扱います。 オームの法則とキルヒホッフの法則から出発し、電圧・電流・電力がどのように決まるか、 そして送電線を流れる「潮流」がどのように生じるかを学びます。 交流回路特有の概念（インピーダンス、有効電力・無効電力）もここで導入します。 この領域は、以降のすべての領域の前提となります。

個別の話に入る前に、まず電力系統の全体像を1枚で見てみましょう。 電気は発電所でつくられ、変圧器で高い電圧に昇圧されて送電線を長距離流れ、 変電所で段階的に電圧を下げながら、最後は需要家（工場・ビル・家庭）へ届きます。 実際には多数の発電所と需要家が母線（接続点）で網の目のようにつながっていますが、 ここではまず「発電 → 送電 → 変電 → 配電 → 需要家」という大きな流れに注目してください。

この図のどこを、本サイトのどの領域で学ぶのかを地図にしておきます。

  • A 系統の基礎：電気の流れ（潮流）がどう決まるか（A-01・A-02・A-03）
  • B 系統混雑：送電線の容量と、流しすぎで起きる「混雑」（B領域）
  • C 系統モデル：系統を計算できる形にモデル化する（C領域）
  • D 混雑計算のアルゴリズム：混雑を解く演算（PTDF・再給電・SCUC/SCED）（D領域）
  • E 電力市場：発電・需要の計画値（同時同量の計画）（E領域）
  • F 中央給電指令所システム：系統全体の監視・制御（F領域）
